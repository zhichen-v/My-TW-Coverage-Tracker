# 1735 - 日勝化

## 業務簡介
**板塊:** Basic Materials
**產業:** Chemicals
**市值:** 2,624 百萬台幣
**企業價值:** 3,227 百萬台幣

日勝化工 (1735，Evermore Chemical) 為日本[[AICA Kogyo]] (持股 50.1%) 之台灣子公司，專精[[PU樹脂]] (聚氨酯樹脂)、[[TPU]] (熱塑性聚氨酯)、接著劑及 UV 硬化塗料製造。產品應用涵蓋鞋材 (供應[[寶成]]、[[豐泰]]等[[Nike]]、[[adidas]]代工廠)、合成皮革 (供應[[三芳]])、綠建材環保塗料 (與母公司[[AICA Kogyo]]共同開發) 及光電抗反射/抗眩 UV 塗料。上游異氰酸酯原料採購自[[BASF]]、[[Covestro]]、[[萬華化學]]。生產基地位於台灣南投、中國及越南。

## 供應鏈位置
**上游 (原料):**
- **異氰酸酯:** [[BASF]]、[[Covestro]]、[[萬華化學]] — [[MDI]]/[[TDI]]
- **技術母公司:** [[AICA Kogyo]] (日本，持股 50.1%)

**中游 (製造):**
- **日勝化** ([[PU樹脂]]/[[TPU]]/接著劑/UV 塗料，台灣/中國/越南)

**下游 (終端):**
- **鞋材:** [[寶成]]、[[豐泰]] — [[Nike]]/[[adidas]]代工廠用 PU 樹脂
- **合成皮革:** [[三芳]] — 合成皮加工
- **綠建材:** 日本/台灣建築市場 — 環保塗料
- **光電:** 面板模組廠 — 抗反射/抗眩 UV 塗料
