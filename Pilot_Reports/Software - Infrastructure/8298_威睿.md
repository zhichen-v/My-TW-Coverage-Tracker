# 8298 - 威睿

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 1,443 百萬台幣
**企業價值:** 1,183 百萬台幣

威睿科技股份有限公司 (8298，Genie Networks) 成立於 1990 年，為台灣網路流量分析與 [[DDoS]] 防護之專業軟體廠商，產品銷售遍及全球逾 40 國。核心產品 GenieATM 為企業級網路流量分析管理系統，結合大數據分析能力優化 IP 網路效能並即時偵測網路攻擊。產品線涵蓋 GenieATM (流量分析)、GenieATM FLB (負載平衡)、GenieAnalytics (深度流量追蹤) 及 [[AI]] 驅動之網路安全解決方案。主要客戶為全球知名電信營運商 ([[ISP]])、政府機構、學術機構及大型企業。公司以高毛利 (逾 90%) 之純軟體授權模式營運。

## 供應鏈位置
**上游 (技術與平台):**
- **伺服器硬體:** 標準化伺服器供應商 — GenieATM 搭載平台
- **雲端運算:** [[AWS]]、[[Microsoft Azure]] — 雲端版本部署
- **AI 技術:** [[AI]] 機器學習模型 — 網路異常偵測

**中游 (網路流量分析軟體):**
- **威睿科技** (GenieATM 網路流量分析、[[DDoS]] 防護、GenieAnalytics 深度追蹤)

**下游 (終端客戶):**
- **電信營運商:** 全球各大 [[ISP]] — 網路流量管理與 DDoS 防護
- **政府機構:** 各國政府 — 網路安全監控
- **學術機構:** 大學與研究機構 — 學術網路管理
- **大型企業:** 跨國企業 — 網路效能優化
