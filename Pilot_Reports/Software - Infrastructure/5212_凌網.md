# 5212 - 凌網

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 1,867 百萬台幣
**企業價值:** 1,494 百萬台幣

凌網科技 (Hyweb) 為台灣專業 **軟體資訊服務商**，深耕政府與金融領域。
主要業務三大支柱：
1.  **電子化政府 (E-Gov)**: 承攬政府大型資訊系統標案 (網站/行政系統)。
2.  **智慧圖書館 (Smart Library)**: **HyRead** 電子書平台 (市佔第一)、圖書館[[自動化]]系統、自助借還書機。
3.  **電子支付/金融**: 協助銀行建置紅利積點系統、電子錢包系統、以及信用卡組織 (Visa/MasterCard) 認證服務。
近年積極推動[[雲端服務]] (SaaS) 與電子書閱讀器硬體銷售。

## 供應鏈位置
*   **中游 (本體)**: **凌網 (5212)** - 軟體開發, 系統整合, 內容平台營運.
*   **上游**:
    *   **內容**: 出版社 (電子書授權).
    *   **硬體**: 閱讀器/借還書機代工廠, [[伺服器]]供應商.
*   **下游**:
    *   **政府**: 中央與地方機關.
    *   **圖書館**: 公共圖書館, 大專院校.
    *   **金融**: [[中信金]], [[國泰金]], [[聯邦銀]].
    *   **消費者**: HyRead 電子書讀者.

## 主要客戶及供應商
### 主要客戶
*   **政府**: 各級政府單位.
*   **圖書館**: 國家圖書館, 各大學圖書館.
*   **金融**: 國內各大發卡銀行, 國際卡組織 ([[Visa]], [[Mastercard]]).

### 主要供應商
*   **出版**: 國內外各大出版社.
*   **硬體**: 振曜 (電子書閱讀器代工 - 推測).
