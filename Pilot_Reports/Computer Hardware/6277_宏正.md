# 6277_宏正

## 業務簡介
**板塊:** Technology
**產業:** Computer Hardware
**市值:** 7,109 百萬台幣
**企業價值:** 6,452 百萬台幣

宏正自動科技股份有限公司 (6277，ATEN) 是全球最大、台灣領先的KVM(多電腦切換器)設備製造商。宏正專注於資訊設備管理、專業影音、智慧製造及綠能電源分配系統的完整解決方案。其「ATEN」品牌產品廣泛應用於大型企業IT基礎設施、廣播電視、醫療設備、工廠[[自動化]]及控制中心。受惠於全球[[資料中心]]建置潮、遠端工作(WFH)及混合辦公模式的興起，宏正的遠端網路控管方案(Over IP KVM)及專業影音分配器需求表現強勁。

## 供應鏈位置
*   **上游**: 微控制器([[MCU]])、[[網通]]晶片、[[PCB]]空板、[[連接器]]、機構件及各類電子被動元件。
*   **中游**: **宏正自動科技** (企業級KVM多電腦切換器、專業影音分配與控制系統研發製造)。
*   **下游**: 企業IT部門、[[資料中心]]管理員、廣播電視/專業影音系統整合商(SI)及製造工廠。

## 主要客戶及供應商
### 主要客戶
*   全球大型企業(如銀行、電信業、知名[[科技巨頭]])、政府機關、教育機構及醫療院所。

### 主要供應商
*   國內外知名IC設計公司、電源供應器廠及機構件沖壓/射出製造商。
