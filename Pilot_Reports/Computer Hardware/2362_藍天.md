# 2362 - 藍天

## 業務簡介
**板塊:** Technology
**產業:** Computer Hardware
**市值:** 22,166 百萬台幣
**企業價值:** 50,421 百萬台幣

藍天電腦 (2362，[[Clevo]]) 成立於 1983 年，總部位於新北市，為台灣最早之筆記型電腦開發先驅，現為全球高階電競筆電 ODM 設計製造領導廠。公司為[[台北雙子星]]集團子公司。

公司專精於超高階電競/工作站等級筆電之準系統 (Barebones) 設計製造，客戶為全球各地客製化電競筆電品牌。產品搭載 [[Intel]] Core Ultra 及 [[NVIDIA]] GeForce RTX 50 系列最新平台，支援高達 192GB DDR5 記憶體。另透過子公司百腦匯經營中國大陸 3C 賣場連鎖事業，以及不動產開發與商場管理業務。

## 供應鏈位置
**上游 (關鍵零組件):**
- **處理器:** [[Intel]] — Core Ultra 系列行動處理器
- **GPU:** [[NVIDIA]] — GeForce RTX 50 系列行動顯卡
- **記憶體:** DDR5 記憶體及 [[SSD]] 固態硬碟模組
- **面板:** 高解析度 UHD+/QHD+ 高刷新率面板

**中游 (電競筆電 ODM 設計製造):**
- **準系統設計:** **藍天** (全球高階電競筆電 Barebones 設計)
- **產品整合:** **藍天** (搭載最新 [[Intel]]/[[NVIDIA]] 平台之準系統)
- **零售通路:** 百腦匯 (中國大陸 3C 賣場連鎖)

**下游 (電競品牌與客製化通路):**
- **電競品牌:** 全球客製化電競筆電品牌 — Barebones 準系統
- **系統整合商:** 歐美日電競/工作站品牌 — ODM 代工
- **消費通路:** 百腦匯中國大陸 3C 賣場終端消費者
