# 3693 - 營邦

## 業務簡介
**板塊:** Technology
**產業:** Computer Hardware
**市值:** 21,882 百萬台幣
**企業價值:** 19,826 百萬台幣

營邦企業 (AIC) 為全球領先的 **伺服器與儲存系統** 解決方案商，已從早期的[[伺服器]]機殼製造轉型為提供 **AI 伺服器準系統 (Barebone)** 與整機櫃解決方案。
公司以高彈性、高密度的設計能力著稱，是大型[[雲端服務]]商 ([[CSP]]) 與企業用戶的重要合作夥伴。
核心優勢與發展亮點：
1.  **AI 伺服器領航者**: 為 **[[AMD]]** 全球首家 MI450 系列 GPU 合作廠，提供專屬機櫃與機殼；並與 **[[NVIDIA]]** 在 [[AI]] Storage 領域維持策略合作。
2.  **散熱技術實力**: 具備先進的液冷 (Liquid Cooling) [[散熱]]架構設計能力，支援 8GPU 等超高密度運算[[伺服器]]，滿足 [[AI]] 運算產生的極大熱管理需求。
3.  **高效能儲存解決方案**: 與 **[[Seagate]]** 及 **[[群聯]] (Phison)** 策略聯盟，開發全快閃[[記憶體]] (All-Flash) 與高密度儲存[[伺服器]]，切入[[資料中心]]與 [[5G]] 邊緣運算市場。
4.  **獲利能力表現**: 憑藉高附加價值的 [[ODM]]/[[OEM]] 業務 (營收佔比已達 77%)，維持優於同業的毛利率與獲利水準。

## 供應鏈位置
*   **上游**: 處理器 ([[Intel]]/[[AMD]]/[[NVIDIA]])、[[記憶體]]模組、電源供應器、金屬機構件。
*   **中游**: **營邦 (3693)** - [[伺服器]]主板開發、系統熱力設計、準系統組裝、整機櫃測試。
*   **下游**:
    *   **二、三線雲端客戶**: 北美及區域型 Tier-2 [[CSP]]。
    *   **軟體定義儲存商**: Media Entertainment, 影像監控系統整合商。
    *   **邊緣運算**: [[5G]] 網絡、自動駕駛與工業 4.0 設施。

## 主要客戶及供應商
### 主要客戶
*   關鍵戰略夥伴：[[AMD]], [[NVIDIA]], [[Intel]]。
*   北美大型雲端與儲存技術應用客戶 (營收 80% 來自美洲)。
*   群[[聯電]]子等 [[AI]] 儲存解決方案合力廠商。
