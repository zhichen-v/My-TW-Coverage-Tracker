# 2395 - 研華

## 業務簡介
**板塊:** Technology
**產業:** Computer Hardware
**市值:** 288,754 百萬台幣
**企業價值:** 271,669 百萬台幣

研華科技 (2395，[[Advantech]]) 成立於 1981 年，總部位於台北，為全球最大[[工業電腦]] ([[IPC]]) 品牌廠暨工業物聯網 ([[IIoT]]) 解決方案領導者。公司產品及服務涵蓋嵌入式電腦、工業自動化、智慧系統、[[Edge AI]] 邊緣運算及雲端服務。

六大事業群：(1) 工業物聯網自動化 (iAutomation)；(2) 嵌入式物聯網 (Embedded-IoT)；(3) 應用運算 (Applied Computing)；(4) 邊緣伺服器 (Edge Server)；(5) 服務物聯網 (iService)；(6) 研華服務+。2025 Q1 營收 173.5 億元，年增 25%，北美及歐洲需求強勁，能源、醫療及半導體為主要成長動能。全球產能布局涵蓋台灣、中國崑山、美國及歐洲。

## 供應鏈位置
**上游 (CPU 與電子元件):**
- **處理器:** [[Intel]]、[[AMD]] — 嵌入式/工業級 CPU 及晶片組
- **GPU:** [[NVIDIA]] — Edge AI 邊緣運算 GPU
- **被動元件:** [[MLCC]]、連接器、電源模組
- **PCB:** 工業級多層 PCB 及散熱模組

**中游 (工業電腦品牌與解決方案):**
- **IPC 品牌:** **研華** (全球最大工業電腦品牌，嵌入式/機架式 IPC)
- **Edge AI:** **研華** (AI 邊緣伺服器及 GPU 運算平台)
- **軟體平台:** **研華** (WISE-IoT 雲端物聯網平台)

**下游 (各產業終端客戶):**
- **工廠自動化:** 全球製造業 — 智慧工廠 IPC/HMI/控制器
- **能源:** 太陽能/風電/儲能 — 邊緣運算監控
- **醫療:** 醫院及醫療設備廠 — 醫療級觸控電腦
- **交通/零售:** 智慧交通及智慧零售解決方案
