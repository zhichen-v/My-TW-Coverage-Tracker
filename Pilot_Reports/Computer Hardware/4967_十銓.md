# 4967 - 十銓

## 業務簡介
**板塊:** Technology
**產業:** Computer Hardware
**市值:** 21,623 百萬台幣
**企業價值:** 23,996 百萬台幣

十銓科技 (Team Group Inc.) 主要從事 **記憶體模組** (DRAM Module) 與 **NAND Flash** 應用產品之研發、製造與銷售。
擁有自有品牌 **[[Team]]** (一般消費)、**[[T-FORCE]]** (電競高效能)、**[[T-CREATE]]** (創作者應用) 與 **[[Team Group Industrial]]** (工控)。
主要產品線：
1.  **記憶體模組**: 桌機/筆電用 RAM、電競超頻記憶體、伺服器記憶體。
2.  **快閃記憶體產品**: 固態硬碟 ([[SSD]]), 記憶卡 (SD/MicroSD/CFexpress), USB 隨身碟。
3.  **工控儲存解決方案**: 應用於博弈、車用、醫療、自動化設備之高耐用度儲存產品。
4.  **行動週邊**: 散熱模組 (SSD Cooler), 充電線材, 集線器。

## 供應鏈位置
*   **中游 (本體)**: **十銓 (4967)** - 記憶體/SSD 模組設計、測試與組裝.
*   **上游**:
    *   **DRAM/NAND 顆粒**: 原廠 ([[Samsung]], [[SK Hynix]], [[Micron]], [[南亞科]]).
    *   **控制晶片 (Controller)**: [[群聯]], [[慧榮]].
*   **下游**:
    *   **通路 (B2C)**: 電商 ([[Amazon]], Newegg), 3C 零售通路 (服務電競玩家, 創作者).
    *   **專案 (B2B)**: 工業電腦 ([[IPC]]), 系統整合商 (SI), OEM 代工客戶.
