# 5403 - 中菲

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 8,366 百萬台幣
**企業價值:** 6,453 百萬台幣

中菲電腦 (5403，DDSC) 成立於 1981 年，為專業資訊系統整合服務商，深耕金融業、證券業及物流業之軟體開發與系統維護。公司以航運海運系統軟體起家，後成功切入金融業信託系統，目前在台灣銀行信託系統市佔率高達約 80%，證券業系統市佔率約 30%，形成極高轉換成本之產業護城河。主要業務涵蓋銀行信託系統、證券交易與帳務平台、貨運承攬管理系統等，同時代理經銷 [[IBM]] 及 [[Cisco]] 等硬體設備。

## 供應鏈位置
**上游 (硬體與軟體平台):**
- **硬體設備:** [[IBM]]、[[Cisco]] (伺服器/網通設備代理)
- **軟體平台:** [[Microsoft]]、[[Oracle]] (資料庫/開發工具)

**中游 (系統整合):**
- **金融/物流 SI:** **中菲電腦** (銀行信託系統、證券交易系統、貨運承攬系統)

**下游 (終端客戶):**
- **銀行業:** 台灣各大公股行庫與民營銀行 (信託系統，市佔約 80%)
- **證券業:** 數十家證券商 (交易與帳務平台，市佔約 30%)
- **物流業:** [[中菲行]] (Dimerco Express，集團關係企業)
- **其他:** [[長榮海運]]、[[新加坡航空]]、國防部、期交所
