# 5211 - 蒙恬

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 730 百萬台幣
**企業價值:** 565 百萬台幣

蒙恬科技 (5211，Penpower) 為智慧型人機介面技術領導者，以「蒙恬筆」手寫輸入與「名片王 (WorldCard)」名片辨識聞名。核心技術涵蓋手寫辨識、OCR 文件辨識、語音辨識與人臉辨識。主要產品線：(1) **消費性電子** — 蒙恬筆、名片王、掃譯筆等硬體搭配辨識軟體；(2) **企業解決方案** — 證件辨識系統、電子簽名 BreezySign、企業名片雲，廣泛應用於銀行無紙化櫃檯與飯店 check-in；(3) **教育科技** — 遠距教學工具 RemoteGo。近年積極轉向 SaaS 雲端服務與 AIoT 應用，並拓展技術授權業務。

## 供應鏈位置
**上游 (零組件與代工):**
- **電子代工:** 國內電子代工廠 (手寫板/掃譯筆硬體製造)
- **晶片供應:** 觸控 IC 與感測器供應商

**中游 (軟體與產品開發):**
- **辨識技術:** **蒙恬** (手寫辨識/OCR/人臉辨識演算法開發、硬體整合設計)

**下游 (終端應用):**
- **通路代理:** [[聯強]]、[[捷元]] (3C 通路代理商)
- **企業客戶:** 銀行 (證件辨識/無紙化櫃檯)、電信業者、飯店業
- **消費市場:** 3C 賣場、電商平台 ([[PChome]]、[[momo]])
