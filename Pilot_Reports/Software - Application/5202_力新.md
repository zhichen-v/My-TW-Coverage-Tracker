# 5202 - 力新

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 1,247 百萬台幣
**企業價值:** 805 百萬台幣

力新國際 (5202，NewSoft) 隸屬於[[台灣鋼鐵集團]]，為台灣知名軟體公司，早年以 OCR (光學字元辨識) 軟體「丹青」與「非常好色」聞名。目前轉型為企業數位轉型服務商：(1) **AI OCR/IDR** — 企業文件辨識與智慧單據輸入系統，支援發票、收據等自動化辨識；(2) **企業解決方案** — BPM 流程管理、雲端辦公室 (Pro OA)、碳盤查系統；(3) **系統整合** — 配合母集團[[台灣鋼鐵集團]]進行資訊系統整合；(4) **AIoT** — 智慧聯網服務。公司以 [[Microsoft]] Azure 及 [[AWS]] 為雲端基礎設施，服務金融、製造及政府客戶。

- **雲端平台:** [[Microsoft]] Azure、[[AWS]]

**中游 (軟體開發):**
- **企業軟體:** **力新** (AI OCR/IDR、BPM 流程管理、碳盤查系統)
