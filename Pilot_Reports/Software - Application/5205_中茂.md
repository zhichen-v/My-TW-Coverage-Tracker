# 5205 - 中茂

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 410 百萬台幣
**企業價值:** 370 百萬台幣

中茂能資 (5205，TGE) 前身為漢平電子轉投資企業，後轉型投入環保綠能領域。目前業務涵蓋：(1) **環境工程** — 水資源處理營運、廢棄物處理與再生利用服務、生質能發電規劃；(2) **通路經銷** — 電子零組件代理、民生消費品代理。公司資本額約 1.35 億元，營運規模較小，業務仍在調整轉型中，主要客戶以政府公部門環保標案為主。

## 供應鏈位置
**上游 (設備與原料):**
- **環保設備:** 水處理設備製造商如 [[東元電機]]、廢棄物處理設備商
- **電子零組件:** 上游零組件原廠如 [[Texas Instruments]]、[[STMicroelectronics]]

**中游 (工程與代理):**
- **環境工程/代理:** **中茂** (水資源營運、廢棄物處理、零組件與消費品代理)

**下游 (終端客戶):**
- **政府標案:** 各縣市[[環境部]]、水利局 (水處理/廢棄物標案)
- **一般市場:** 零售通路 (民生消費品)

## 主要客戶及供應商
### 主要客戶
- **政府公部門:** 各縣市[[環境部]]、水利局 (環保工程標案)
- **生質能:** [[台電]] — 生質能發電躉售
- **一般市場:** 零售通路 (民生消費品經銷)

### 主要供應商
- **環保設備:** 水處理與廢棄物處理設備製造商如 [[東元電機]]
- **電子零組件:** 上游零組件原廠如 [[Texas Instruments]] (代理經銷)
