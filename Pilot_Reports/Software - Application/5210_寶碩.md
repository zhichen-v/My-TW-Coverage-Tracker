# 5210 - 寶碩

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 1,833 百萬台幣
**企業價值:** 1,498 百萬台幣

寶碩財務科技 (5210，Apex) 為台灣專業金融資訊系統開發商。主要業務：(1) **金融工程** — 銀行/證券之財務交易系統、風險管理系統、資產負債管理系統 (ALM)，技術強項在於複雜的財務模型運算與即時交易系統架構；(2) **系統整合** — 視訊會議系統、數位金融學院平台；(3) **綠能事業** — 近年跨足太陽能發電案場投資與電力交易市場。客戶涵蓋國內各大金控、銀行與證券商，以 [[Microsoft]] 及 [[Oracle]] 為核心技術平台。

## 供應鏈位置
**上游 (技術平台):**
- **軟體平台:** [[Microsoft]]、[[Oracle]] (資料庫/開發平台)
- **資訊源:** [[臺灣證券交易所]]、[[櫃買中心]] (行情資料)
- **太陽能模組:** 國內太陽能 EPC 廠商 (綠能業務)

**中游 (金融軟體):**
- **金融系統開發:** **寶碩** (財務交易系統、風控系統、ALM、視訊平台)

**下游 (金融機構):**
- **銀行:** [[兆豐金]]、[[中信金]]、各大公股行庫
- **證券:** 國內券商 (交易系統/風控系統)
- **政府/教育:** 視訊系統客戶

## 主要客戶及供應商

- **銀行業:** [[兆豐金]]、[[中信金]]、各大公股行庫 (風控/交易/ALM 系統)

- **政府/教育:** 視訊系統與數位學院平台
