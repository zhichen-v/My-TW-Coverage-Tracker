# 6882_甲尚

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 313 百萬台幣
**企業價值:** 122 百萬台幣

甲尚科技 (6882，Reallusion) 為全球角色動畫與多媒體軟體工具領域之台灣領導企業，成立於 1982 年，總部位於新北市。公司專注於降低高階動畫製作門檻，核心產品 [[iClone]] 為實時 3D 動畫軟體，搭配 Character Creator 角色建模工具與 Cartoon Animator 2D 動畫軟體，讓獨立影片創作者或小型工作室也能完成電影級別的角色動畫。公司與 [[NVIDIA]]、[[Epic Games]] (Unreal Engine)、[[Unity]] 等大廠建立技術與行銷夥伴關係，並整合 [[OptiTrack]]、[[Xsens]]、[[Rokoko]] 等穿戴式動捕設備。甲尚亦營運全球數位內容資產平台，銷售服裝、動作庫等海量素材。

## 供應鏈位置
**上游 (技術與內容):**
- **GPU/AI 運算:** [[NVIDIA]] (GPU 加速渲染與 AI 技術合作)
- **遊戲引擎:** [[Epic Games]] (Unreal Engine)、[[Unity]]
- **動捕設備:** [[OptiTrack]]、[[Xsens]]、[[Rokoko]] (穿戴式動作捕捉)
- **數位內容:** 全球獨立數位藝術家 (3D 建模/動作素材供應)

**中游 (軟體開發與平台):**
- **動畫軟體:** **甲尚** (iClone、Character Creator、Cartoon Animator、數位資產平台)

**下游 (終端應用):**
- **影視動畫:** 全球動畫電影製片廠與影視工作室
- **遊戲開發:** 頂尖與獨立遊戲開發商
- **虛擬網紅:** VTuber 虛擬主播經濟
- **教育訓練:** 各級教育機構與企業培訓
