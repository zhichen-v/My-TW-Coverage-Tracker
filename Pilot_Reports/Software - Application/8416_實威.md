# 8416 - 實威

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 4,754 百萬台幣
**企業價值:** 4,282 百萬台幣

實威國際 (8416，SolidWizard) 成立於 1997 年，為法國 [[Dassault Systemes]] 在亞洲最大專業代理商。公司代理銷售 [[SOLIDWORKS]] 3D CAD 設計軟體及 [[3DEXPERIENCE]] 雲端協同設計管理平台，涵蓋 CAD、CAE 驗證分析 ([[SIMULIA]])、PDM/PLM 管理軟體等全方位產品線。此外，智慧製造業務提供 [[Markforged]] 與 [[3D Systems]] 3D 列印機、Artec 3D 逆向掃描及 [[Universal Robots]] 協作機器人。至今在台灣及中國地區已輔導超過 10,000 家企業與教育單位。

## 供應鏈位置
**上游 (軟體與硬體原廠):**
- **3D CAD/PLM:** [[Dassault Systemes]] ([[SOLIDWORKS]]、[[3DEXPERIENCE]]、[[SIMULIA]])
- **3D 列印:** [[Markforged]]、[[3D Systems]]
- **協作機器人:** [[Universal Robots]]
- **3D 掃描:** Artec、SCANTEC

**中游 (代理與顧問):**
- **軟硬體代理:** **實威國際** (軟體銷售、顧問導入、教育訓練、售後服務)

**下游 (終端應用):**
- **製造業:** 國內機械、電子、汽車零組件等製造業 R&D 部門
- **教育機構:** 各級大專院校與技職體系
- **醫療/航太:** 高階設計與模擬分析應用
