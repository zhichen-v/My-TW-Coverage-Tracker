# 6902_GOGOLOOK

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 2,701 百萬台幣
**企業價值:** 2,683 百萬台幣

走著瞧 (6902，Gogolook) 為國際知名之「信任科技 (TrustTech)」AI 軟體服務公司，從台灣國民 App「[[Whoscall]]」起家。透過用戶回報與 AI 大數據分析，累積全球龐大的電話識別與防詐騙惡意資料庫。公司業務延伸至企業端防詐 ([[ScamAdviser]] 全球假網址與商譽風險偵測)、個人端防詐 (美玉姨查核系統) 及金融科技媒合服務 (袋鼠金融 Roo.Cash)。公司深受 [[NAVER]] ([[LINE]] 母公司) 資金挹注與技術肯定，亦與台灣警政署、韓國金融監督院、泰國警方及馬來西亞州政府等多國安保單位合作。

## 供應鏈位置
**上游 (雲端與數據):**
- **雲端平台:** [[AWS]]、[[Google Cloud]] (公有雲運算與儲存)
- **電信 API:** 各國基礎電信營運商 (API 串接線路)
- **策略投資:** [[NAVER]] ([[LINE]] 母公司，早期最大外部控股)

**中游 (TrustTech 平台):**
- **防詐/信任科技:** **Gogolook** ([[Whoscall]] 來電辨識、[[ScamAdviser]] 企業防詐、袋鼠金融媒合)

**下游 (終端客戶):**
- **消費者:** 全球智慧手機 App 訂閱用戶
- **企業端:** [[凱基證券]]、[[中信金]]等大型金控 (防詐合作)
- **政府端:** 台灣警政署、韓國金融監督院、泰國/馬來西亞警方
- **電信合作:** [[Globe Telecom]] (菲律賓)、[[StarHub]] (新加坡)
