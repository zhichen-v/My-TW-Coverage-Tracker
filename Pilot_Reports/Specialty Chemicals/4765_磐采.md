# 4765 - 磐采

## 業務簡介
**板塊:** Basic Materials
**產業:** Specialty Chemicals
**市值:** 566 百萬台幣
**企業價值:** 812 百萬台幣

磐采科技 (4765，Pancolour Ink) 為特殊 UV 油墨/塗料製造商 (逾 25 年經驗)，品牌 PANOVA/PANCURE/PANOLEN/RADITHO。產品涵蓋 UV 網版印刷油墨 (光碟/卡片/包裝)、UV 塗料/接著劑及[[3D 列印]]樹脂。正積極轉型為電子零組件材料商 — 開發[[MLCC]]用導電漿料 (銅漿/銀漿，用於內電極/端電極)，潛在客戶涵蓋[[國巨]]、[[華新科]]、日本[[村田]]等[[MLCC]]大廠。另開發[[太陽能]]電池導電漿料。

## 供應鏈位置
**上游 (原料):**
- **樹脂/顏料:** 特殊樹脂及顏料供應商
- **金屬粉末:** 銅粉/銀粉 — 導電漿料用

**中游 (製造):**
- **磐采** (UV 油墨/塗料；[[MLCC]]導電漿料；[[3D 列印]]樹脂)

**下游 (終端):**
- **[[MLCC]] (潛在):** [[國巨]]、[[華新科]]、[[村田]] — 導電漿料
- **太陽能:** [[太陽能]]電池 — 導電漿料
- **印刷:** 特殊印刷/包裝 — UV 油墨

## 主要客戶及供應商
### 主要客戶
- **[[MLCC]] (潛在):** [[國巨]]、[[華新科]]、[[村田]] — 導電漿料
- **太陽能:** [[太陽能]]電池導電漿料
