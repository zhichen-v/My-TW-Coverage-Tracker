# 6899_創為精密

## 業務簡介
**板塊:** Technology
**產業:** Electronic Components
**市值:** 2,149 百萬台幣
**企業價值:** 1,543 百萬台幣

創為精密 (6899，[[TES Touch Embedded Solutions]]) 專攻工控、醫療與交通等特殊利基型觸控顯示器市場。有別於消費性電子觸控面板廠，創為專注於需面臨極端環境的觸控解決方案。

業務結構與動能：(1) 工控觸控面板 — 防水、抗油汙、耐震動之工業級觸控顯示器，供應全球高階[[工業電腦]]製造商；(2) 醫療觸控 — 通過 [[UL]] 醫療安規認證之手術室與監測儀器用觸控面板；(3) 車用觸控 — 耐高溫高震的車載觸控顯示器，打入頂級豪華汽車 Tier 1。公司提供從觸控面板、觸控控制器 IC、光學貼合到開放式機架螢幕的一站式客製化方案。

## 供應鏈位置
**上游 (材料與元件):**
- **玻璃基板:** 特殊鍍膜玻璃基板 (防爆/抗反射)
- **ITO 膜:** 透明導電膜供應商
- **觸控 IC:** 觸控控制晶片設計廠
- **[[OCA]] 光學膠:** 光學級貼合用光學膠

**中游 (觸控面板製造):**
- **工控觸控:** **創為** (工業級觸控顯示面板)
- **醫療觸控:** **創為** ([[UL]] 醫療認證觸控面板)
- **車用觸控:** **創為** (車載觸控顯示器)

**下游 (終端廠商):**
- **[[工業電腦]]:** 全球高階 IPC 製造商 — 工業觸控面板
- **醫療器材:** 歐美醫療手術器械原廠 — 醫療觸控顯示
- **汽車:** 豪華汽車 Tier 1 — 車載觸控面板
- **航太海運:** 航太海運儀表板廠 — 特殊環境觸控
