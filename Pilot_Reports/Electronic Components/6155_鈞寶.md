# 6155_鈞寶

## 業務簡介
**板塊:** Technology
**產業:** Electronic Components
**市值:** 3,895 百萬台幣
**企業價值:** 3,773 百萬台幣

鈞寶電子 (6155) 為專業被動電子零組件製造商，聚焦於電磁干擾 (EMI) 防治與電源管理解決方案。

主要產品：(1) EMI 鐵氧體磁性材料 — 各式鐵氧體磁芯、共模抗流圈 (Common Mode Choke)；(2) 電感器與磁珠 — 積層晶片電感、晶片磁珠；(3) [[LTCC]] 晶片電感 — 低溫共燒陶瓷技術，已成功打入 [[5G]] 供應鏈，供應日、歐、美及中國客戶；(4) 車用 EMI 元件 — 受惠於[[電動車]]電子化趨勢，車用需求持續成長。近年受惠於 [[5G]] 通訊、[[AI 伺服器]]及[[電動車]]電子化需求帶動。

## 供應鏈位置
**上游 (原材料):**
- **氧化鐵粉:** 鐵氧體磁性材料原料
- **銀漿:** 貴金屬銀漿 — 關鍵材料，占成本比重高
- **金屬材料:** 鎳、銅金屬材料供應商

**中游 (被動元件製造):**
- **EMI 元件:** **鈞寶** (鐵氧體磁芯、共模抗流圈)
- **電感/磁珠:** **鈞寶** (積層晶片電感、晶片磁珠)
- **[[LTCC]]:** **鈞寶** ([[5G]] 用 [[LTCC]] 晶片電感)

**下游 (終端應用):**
- **[[AI 伺服器]]:** [[AI 伺服器]]代工廠 — EMI 濾波與電感
- **主機板:** 主機板製造廠 — EMI 元件
- **[[5G]] 通訊:** [[5G]] 基地台與設備商 — [[LTCC]] 晶片電感
- **車用:** 日系、歐系車用電子客戶 — 車用 EMI 元件

## 主要客戶及供應商
### 主要客戶
- **[[AI 伺服器]]:** 伺服器代工廠 — EMI 濾波與電感元件
- **主機板:** 主機板製造廠 — EMI 防治元件
- **[[5G]] 通訊:** [[5G]] 基地台設備商 — [[LTCC]] 晶片電感
- **車用電子:** 日系、歐系車用電子 Tier 1 客戶 — 車用 EMI 元件
- **[[網通]]設備:** 網通設備代工廠 — EMI 元件

- **金屬材料:** 鎳、銅金屬材料供應商
